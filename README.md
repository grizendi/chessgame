# Simple 2D Chess Game

This project contains a minimal implementation of a chess game using HTML, CSS and JavaScript. The board is rendered with basic styling and the pieces use Unicode chess characters.

## Running the Game

Open `src/index.html` in any modern web browser. No additional dependencies or build steps are required.

## Features

- Standard 8x8 chess board
- Click a piece to see its possible moves
- Basic movement rules for all pieces
- Automatic pawn promotion to queen when reaching the last rank

The game currently supports two players taking turns on the same device. Castling and en passant are not implemented.
